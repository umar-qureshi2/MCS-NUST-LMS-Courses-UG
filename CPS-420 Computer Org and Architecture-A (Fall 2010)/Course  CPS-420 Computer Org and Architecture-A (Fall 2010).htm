Course Information

Course Objective

The objective of this course is to study computer architecture design by examining architectural concepts with consideration of performance, usability, reliability, power management etc.

This course covers a number of topics such as Instruction Set Architecture, Pipeline Microprocessor, Memory Organization and Management, I/O Management, Interrupt Handler etc to give deep insight about the computer architecture to the students.
